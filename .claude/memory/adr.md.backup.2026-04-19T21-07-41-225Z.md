# Architecture Decision Records (ADR)

Registro delle decisioni architetturali significative.
A differenza di `decisions.md` (note rapide), le ADR seguono un formato più formale
per decisioni ad alto impatto o difficilmente reversibili.

Usa `/remember` per note veloci. Usa questo file per decisioni che cambiano il progetto.
